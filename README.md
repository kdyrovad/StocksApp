# **StocksApp**
StocksApp - an application that makes it easy to follow state of top cryptocurrencies
## **Overview**
+ Design pattern: MVP(Model-View-Presenter)
+ Local data manager: FileManager
+ UI: Programmatical
+ Data provided by: coingecko
